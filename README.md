# Python 文件批量处理自动化脚本

## 项目介绍
基于 Python os 模块实现的文件批量处理自动化脚本，解决日常文件遍历、重命名、格式筛选等重复劳动场景，提升工作效率。

## 核心功能
- 遍历指定目录下所有文件，自动筛选目标格式（如 .txt/.log）
- 支持批量重命名（添加前缀+时间戳）
- 增加目录存在性检查，避免目录不存在报错
- 异常处理：防止文件被占用/权限不足导致脚本崩溃
- 统计处理成功/失败数量，方便核对结果
